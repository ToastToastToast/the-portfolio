If you want a grid that wraps to a column on smaller screens), set the flex-direction for MAIN to row and flex-wrap to wrap, then change justify-content to center your items. Next, set an initial width for each SECTION in the flex property (eg 'flex: 0 1 300px;' instead of 'flex: 0 1 auto').


^ you can customize this more using media queries

Note that the images will scale up and down to the max width of the section, so you want to make sure your image resolution is as least as wide as your section width.
