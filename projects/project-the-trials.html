The Trials


Description

The Trials is a 2d platformer pixel art game for a game studies class at San Jose State University. This game was created alongside Richer Fang and Evelyn Palmeno. This was my first game working with Unity.
Spring 2023


Story

The Trials is about a fox who must past the challenge set by their clan. The young fox must prove themselves worthy of becoming an adult fox and be treated with respect. Will the young fox succeed?
